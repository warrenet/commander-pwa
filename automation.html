Use these Deep Links to control Commander from MacroDroid, Tasker, or Shortcuts.

Add Task (Silent)

Adds a task to the Inbox without opening the full UI or playing sounds. Great for background triggers.

/?command=add&text=Buy Milk&silent=true

Add Task (High Priority)

Adds a high-priority task with tag #priority.

/?command=add&text=Urgent Fix&priority=high

Quick Log

Append a timestamped entry to your logs.

/?command=log&text=Arrived at Gym&silent=true

Focus Mode

Activate Focus Mode directly.

/?command=focus-mode

Toggle Theme

Switch between Light/Dark mode (e.g. at sunset/sunrise).

/?command=toggle-theme&silent=true

Open Specific View

Open different sections directly.

/?view=shipToday
